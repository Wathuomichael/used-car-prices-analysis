Fitting the model

Removing outliers and Car name column.

prices <- subset(prices, Selling_Price < 20, select = - Car_Name)

First we split the data with a split ratio of 70%

set.seed(0)
data.split <- sample.split(prices, SplitRatio = 0.7)
train <- subset(prices, data.split == T)
test <- subset(prices, data.split == F)

We fit the model using the training set.

lm.model <- lm(log(Selling_Price) ~ ., train)
summary(lm.model)

Prediction

lm.preds <- exp(predict(lm.model, test))
lm.rmse <- rmse(test$Selling_Price, lm.preds)
lm.mae <- mae(test$Selling_Price, lm.preds)

RMSE and MAE are relatively low which indicates the model is able to accurately predict selling price.

ggplot(test) + geom_line(aes(x = Selling_Price, y = lm.preds), colour = 'red') + 
  theme_minimal() + 
  labs(title = 'Predicted vs Actual values', x = 'Actual Values', y = 'Predicted Values')

This plot shows the model predictions against actual values of selling price. The model appears to be very accurate at low values of selling price and gradually becomes less accurate as the selling price increases.
